# Calculator
A simple basic calculator which runs on console.

Features:
1) Runs on cmd and console.
2) Used for basic calculations suchas:
    a) Addition
    b) Subtraction
    c) Multiplication
    d) Division
    
Example:
Enter the Operator '+','-','*','/' : 
+
Enter num1: 2
Enter num2: 6
The result is: 8.0
